## 짧은 설명 (80자)
엔카 매물의 사고이력·진단·가격을 12가지 규칙으로 자동 평가하는 종합 점수 도구

## 긴 설명 (4000자 이내)

```
AutoVerdict는 엔카(encar.com) 중고차 매물을 자동으로 분석해
종합 점수와 판정 결과를 보여주는 평가 도구입니다.

복잡한 사고이력, 진단 결과, 정비 기록을 일일이 살펴볼 필요 없이
URL 한 번이면 위험·주의·양호 항목을 한눈에 정리해 드립니다.


◆ 주요 기능

▷ URL 입력 한 번으로 자동 분석
   엔카 매물 페이지 주소를 붙여넣거나, 공유 버튼으로 AutoVerdict를
   선택하면 즉시 분석이 시작됩니다.

▷ 12가지 규칙 기반 종합 점수
   사고이력, 침수/완파 여부, 주행거리 대비 연식, 정비 기록 등을
   가중치 기반으로 평가하여 0~100점 종합 점수를 산출합니다.

▷ 위험 · 주의 · 양호 한눈에
   매물의 핵심 신호를 빨강·주황·초록 카테고리로 분류해
   놓치기 쉬운 위험 요소를 즉시 확인할 수 있습니다.

▷ 매물 저장 & 2대 비교
   마음에 드는 매물을 저장하고, 두 매물을 나란히 비교하여
   점수·연식·주행거리·가격·진단 결과까지 한 화면에서 분석합니다.

▷ 24시간 캐시
   분석 결과는 24시간 동안 캐시되어 같은 매물을 다시 열 때
   네트워크 사용 없이 즉시 표시됩니다.


◆ 안전한 데이터 처리

· 모든 데이터는 사용자의 기기에만 저장됩니다
· 자체 서버가 없으며, 외부로 개인정보를 전송하지 않습니다
· 위치·카메라·연락처 등 민감 권한을 요청하지 않습니다
· 광고 식별자(AdMob)만 사용하며, 광고 추적은 기기 설정에서
  언제든 거부할 수 있습니다


◆ 이런 분께 추천합니다

· 중고차 구매를 처음 알아보고 있어 무엇을 확인해야 할지 막막한 분
· 여러 매물을 비교하고 싶지만 핵심 지표를 빠르게 정리하고 싶은 분
· 사고이력서·진단표를 어떻게 읽어야 하는지 익숙하지 않은 분


◆ 면책

본 앱은 엔카(encar.com)의 공식 서비스가 아니며, SK엔카닷컴(주)과
일체의 제휴·승인 관계가 없습니다. 본 앱이 산출하는 점수와 판정은
알고리즘 기반 참고 정보이며, 실제 차량 상태에 대한 법적 보증이
아닙니다. 최종 구매 결정은 반드시 사용자 본인의 판단 하에 진행해
주시기 바랍니다.
```
