# AutoVerdict 개인정보처리방침

**최종 업데이트:** 2026년 6월 16일
**적용 대상:** AutoVerdict Android 앱 (이하 "본 앱")
**문의:** pouu69@gmail.com

---

## 1. 개요

AutoVerdict는 사용자가 입력한 엔카(encar.com) 중고차 매물의 공개 정보를 분석하여 점수와 평가 결과를 제공하는 Android 애플리케이션입니다. 본 방침은 본 앱이 처리하는 데이터의 범위와 사용 목적, 보호 조치를 설명합니다.

본 앱은 **엔카(encar.com) 또는 SK엔카닷컴(주)와 무관**한 비공식 도구이며, 공개된 매물 정보만을 사용자의 요청에 따라 조회·분석합니다.

---

## 2. 수집·처리하는 정보

### 2.1 사용자가 입력하거나 공유하는 정보
- 엔카 매물 페이지 URL

### 2.2 본 앱이 자동으로 조회하는 공개 정보
- 사용자가 입력한 URL의 매물에 대한 다음 공개 정보:
  - 차량 모델, 연식, 주행거리, 가격
  - 사고 이력, 진단 정보, 정비 기록
  - 매물 사진 썸네일 URL

### 2.3 광고 식별자 (Google AdMob)
- Google AdMob SDK가 사용하는 광고 식별자(AAID, Android Advertising ID)
- 광고 노출·측정 목적으로 Google에 전송됩니다
- 기기 설정에서 광고 추적 거부 또는 광고 ID 재설정이 가능합니다

### 2.4 본 앱이 수집하지 않는 정보
- 사용자의 이름·전화번호·이메일·계정 정보
- 위치 정보 (GPS·IP 위치)
- 사진·연락처·캘린더 등 기기 내 개인 데이터
- 결제 정보

---

## 3. 정보 이용 목적

| 정보 | 목적 |
|---|---|
| 매물 URL | 분석 대상 매물 식별 |
| 매물 공개 정보 | 12가지 평가 규칙에 따른 점수·판정 산출 |
| 광고 ID | AdMob을 통한 광고 노출 및 통계 |

---

## 4. 데이터 저장·보관

### 4.1 저장 위치
모든 분석 데이터는 **사용자의 기기 내부 데이터베이스(Room)에만 저장**되며, 본 앱은 자체 서버를 운영하지 않습니다.

### 4.2 보관 기간
- **분석 캐시:** 분석 완료 시점 기준 24시간 후 자동 만료
- **저장한 매물:** 사용자가 명시적으로 저장한 매물은 삭제 전까지 보관
- **광고 ID:** Google AdMob 정책에 따름 ([AdMob 개인정보처리방침](https://policies.google.com/privacy))

### 4.3 삭제
- 캐시 데이터: 설정 화면 → "캐시 삭제"
- 저장한 매물: 저장 목록 화면에서 개별 삭제
- 전체 데이터: 앱 삭제 시 모든 데이터가 함께 제거됩니다

## 5. 네트워크 통신

본 앱은 다음 외부 호스트와 통신합니다:

| 호스트 | 목적 | 데이터 |
|---|---|---|
| `encar.com`, `fem.encar.com`, `api.encar.com` | 매물 공개 정보 조회 | 사용자가 입력한 URL의 매물 ID |
| Google AdMob (`googleadservices.com` 등) | 광고 노출 | 광고 ID, 디바이스 정보 |

본 앱은 사용자의 개인정보를 외부 서버로 전송하지 않습니다.

## 6. 권한 사용

| 권한 | 용도 | 필수 여부 |
|---|---|---|

| `com.google.android.gms.permission.AD_ID` | Google AdMob 광고 ID 접근 | 필수 |

본 앱은 위치·카메라·연락처·저장공간 등 민감 권한을 요청하지 않습니다.

## 7. 제3자 SDK

본 앱은 다음 제3자 SDK를 사용합니다:

### Google AdMob (Play Services Ads)
- **목적:** 배너 및 전면 광고 표시
- **데이터:** 광고 ID, 일반 기기 정보
- **정책:** [Google 개인정보처리방침](https://policies.google.com/privacy)

본 앱은 위 외의 제3자에게 사용자 데이터를 제공·판매하지 않습니다.

## 9. 사용자의 권리

사용자는 언제든지 다음과 같이 데이터를 통제할 수 있습니다:

- **조회·삭제:** 앱 내 저장 목록·캐시 화면에서 직접 관리

- **앱 삭제:** 앱을 제거하면 모든 로컬 데이터가 즉시 삭제됩니다
